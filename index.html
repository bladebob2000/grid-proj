Is Dark Humor Morally Permissible?

Dark humor shows up in serious and stressful situations. Some people believe it helps others cope, while others think it crosses a moral line. This project explores whether dark humor can be morally acceptable and under what conditions.

What this project is asking

This project is not about whether a dark joke is funny. It is about what the joke does to people. The same joke can be relief in one context and harm in another context.
